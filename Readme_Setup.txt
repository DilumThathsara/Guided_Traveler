1.	Customer and Organizer Mobile Application

	The Customer Mobile App's source code is in the "Mobile Application/User/ Source_Code" folder.
	The Organizer Mobile App's source code is in the "Mobile Application/ Organizer / Source_Code" folder.


1.1	Usage (Install)

Step 1: First, the source code folder in VS code or Android Studio
Step 2: Run the "flutter clean" command. 
Step 3: Run the "flutter pub get" command.
Step 4: Launch the emulator with Android version above 9.0.
Step 5:  Run the "flutter run" command.
Step 6:  Mobile App will launch in the emulator.


1.2	APKs

	The Customer Mobile App's source code is in the "Mobile Application/User/ APK_File" folder.
	The Organizer Mobile App's source code is in the "Mobile Application/ Organizer / APK_File " folder.




2.	Web application

	The Organizer Mobile App's source code is in the "Web Application/ Source_Code " folder.

2.1	Usage (Install)

Step 1: First, the source code folder in VS code
Step 2: Run the "npm run serve" command. 
Step 3: Open the web browser and type “https://localhost:4000”  and press enter
Step 4: The web application will preview.
